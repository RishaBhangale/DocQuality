# DocQuality - Unstructured Data Quality

Frontend application for the Document Quality Intelligence platform. Evaluates unstructured document quality across multiple data quality dimensions.

## Tech Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS
- Recharts
- shadcn/ui components

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
